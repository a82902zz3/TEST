# TEST
第三題:
因為標籤都是錯誤的所以
如果是鉛筆標籤他可能是只裝原子筆或是混和。
如果是原子筆標籤他可能是只裝鉛筆或是混和。
如果是混和標籤他可能是只裝鉛筆或是只裝原子筆。

所以當選擇混和標籤的袋子拿取裡面的筆時可以確定混和標籤的袋子是鉛筆或是原子筆的袋子。

若混和標籤的袋子為只裝鉛筆的袋子，那貼上原子筆標籤的袋子必為混和，因為貼上原子筆標籤的袋子內不會是只裝原子筆。
所以
混和標籤為只裝鉛筆的袋子
原子筆標籤為混和的袋子
鉛筆標籤為只裝原子筆的袋子

反之亦然
若混和標籤的袋子為只裝原子筆的袋子則

混和標籤為只裝原子筆的袋子
原子筆標籤為只裝的鉛筆的袋子
鉛筆標籤為混和的袋子
